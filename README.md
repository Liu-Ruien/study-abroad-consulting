# 出国咨询信息整理平台

这是一个个人开发的出国咨询信息整理网站，主要面向准备出国留学、生活、办理签证，以及正在规划海外发展路线的人群。

当前项目已经完成内容网站 MVP、产品化 UI 优化、GitHub 上传、Vercel 部署，并新增了第三阶段的出国路线规划页面。

------

## 在线访问

网站已部署到 Vercel：

[https://study-abroad-consulting.vercel.app](https://study-abroad-consulting.vercel.app/)

------

## 项目定位

本项目最初是一个出国咨询信息整理网站，当前正在逐步升级为：

> 面向中国大陆用户的出国路线规划与真实经验整理平台。

项目希望解决的问题包括：

- 出国信息分散，不方便系统查看
- 新手用户不知道从哪里开始准备
- 留学、签证、生活、预算、国家选择等内容需要主题化整理
- 用户需要根据预算、学历、专业、语言能力和目标国家获得初步路线建议
- 后期可以扩展为 AI 问答、真实经验社区、付费咨询和预约服务平台

## 当前已实现功能

### 首页

- 网站介绍
- 内容分类入口
- 推荐文章展示
- 跳转到全部文章页
- 跳转到路线规划页

### 文章系统

- 文章列表页
- 文章详情页
- 支持根据文章 slug 访问单篇文章

示例：

```text
/articles/nz-study-suitable
```

### 分类系统

当前支持四个分类：

- 日本留学
- 新西兰留学
- 签证指南
- 海外生活

分类页示例：

```text
/categories/japan-study
/categories/new-zealand-study
/categories/visa-guide
/categories/overseas-life
```

### 搜索与筛选

文章列表页支持：

- 关键词搜索
- 分类筛选
- 搜索结果数量提示
- 无结果提示

### 出国路线规划

第三阶段新增 `/plan` 出国路线规划页面。

当前支持用户填写：

- 年龄
- 学历
- 专业方向
- 预算区间
- 语言能力
- 国家 / 地区偏好
- 是否希望打工
- 是否希望长期留下
- 是否接受低预算路线

系统会根据本地规则推荐 3 条出国路线，并展示：

- 推荐路线名称
- 适合人群
- 路线优势
- 风险提醒
- 下一步建议
- 推荐阅读文章

当前路线规划功能是 MVP 原型，不接真实 AI，不接数据库。

推荐结果基于本地规则生成，主要用于帮助用户建立初步方向，不构成签证、移民、法律或就业承诺。

## 核心模块说明

### 文章数据模块

文件：

```text
lib/articles.ts
```

作用：

- 存放文章数据
- 存放分类数据
- 提供分类名称查询方法

输入：

- 文章 slug
- 分类 slug

输出：

- 文章内容
- 分类名称
- 分类文章列表

------

### 文章列表模块

文件：

```text
app/articles/page.tsx
components/ArticleSearch.tsx
```

作用：

- 展示全部文章
- 支持关键词搜索
- 支持分类筛选

输入：

- 用户输入的搜索关键词
- 用户选择的分类

输出：

- 过滤后的文章列表

------

### 文章详情模块

文件：

```text
app/articles/[slug]/page.tsx
```

作用：

- 根据 URL 中的 slug 展示对应文章详情

输入：

```text
/articles/文章slug
```

输出：

- 文章标题
- 文章简介
- 文章分类
- 文章正文

------

### 分类页面模块

文件：

```text
app/categories/[category]/page.tsx
```

作用：

- 根据分类 slug 展示对应分类下的文章

输入：

```text
/categories/分类slug
```

输出：

- 分类名称
- 当前分类下的文章数量
- 当前分类下的文章列表

------

### 出国路线规划模块

文件：

```text
app/plan/page.tsx
lib/plan-routes.ts
```

作用：

- 收集用户的基础出国需求
- 根据预算、语言能力、目标地区、打工意愿等条件进行本地规则匹配
- 输出适合用户参考的出国路线建议

输入：

- 年龄
- 学历
- 专业方向
- 预算区间
- 语言能力
- 国家偏好
- 是否希望打工
- 是否希望长期留下
- 是否接受低预算路线

输出：

- 推荐路线
- 适合原因
- 优势
- 风险提醒
- 下一步建议
- 推荐阅读文章
