## 中文简历版

### 150 字版

个人开发的出国信息整理作品集。使用 Next.js、React、TypeScript 实现文章系统、基于本地规则的 `/plan` 路线规划、mock 版 `/ai` 问答 MVP 及多语言 Case Study 展示页。负责需求拆解、页面实现、规则逻辑、UI 组件、文档整理与 Vercel 部署。不接真实 AI API 与数据库，重点展示前端工程与产品边界设计能力。

### 300 字版

个人作品集项目，面向日本 IT 求职场景，解决「出国信息分散、难以结构化理解」的问题。技术栈为 Next.js App Router、React、TypeScript、Tailwind CSS，部署于 Vercel。

主要功能包括：内容文章系统（分类、搜索、多语言详情）；`/plan` 路线规划（预算、语言、目标、风险等输入，本地规则引擎生成可解释推荐）；`/ai` 本地问答 MVP（mock 结构化回答，明确不接真实 API）；`/case-study` 面试讲解型项目展示页（中文 / 日本語 / English）。

个人独立负责页面结构设计、组件拆分、lib/plan 与 lib/ai 业务逻辑、多语言文案、README 与面试文档、GitHub 版本管理与线上部署。项目不接数据库、登录与付费，强调可讲解、可演示、可迭代的工程表达。

## 日本語履歴書版

### 短文版（履歴書・職務経歴書向け）

個人開発の留学・海外生活情報整理ポートフォリオ。Next.js、React、TypeScript で記事システム、ローカルルールによる `/plan` 進路プランニング、mock ベースの `/ai` 問答 MVP、多言語 Case Study ページを実装。要件整理から UI 実装、ロジック分離、ドキュメント整備、Vercel デプロイまで個人で担当。商用サービスではなく、フロントエンド実装力とプロダクト境界設計を示す目的。

### 詳細版（Wantedly / Green / 面接資料向け）

**プロジェクト名：** 留学・海外生活 情報整理プラットフォーム（個人ポートフォリオ）

**概要：** 日本 IT 就職向けに作成した個人開発プロジェクト。留学・海外生活に関する情報を、記事、ルート計画、ローカル AI 問答 MVP、ケーススタディページとして整理・展示する。

**使用技術：** Next.js（App Router）、React、TypeScript、Tailwind CSS、Git / GitHub、Vercel

**主な機能：**
- 記事システム（カテゴリ、検索、多言語表示）
- `/plan`：予算・語学・目標・リスクを入力し、ローカルルールで説明可能なルート提案
- `/ai`：mock による構造化問答（本番 AI API 未接続）
- `/case-study`：面接向けプロジェクト説明（中文 / 日本語 / English）

**担当範囲：** ページ設計、コンポーネント実装、`lib/plan` / `lib/ai` ロジック分離、多言語対応、README・面接資料作成、デプロイ

**補足：** DB・ログイン・課金は未実装。ビザ・就業の保証は提供しない情報整理・ポートフォリオ展示が目的。

## English Resume Version

### Short Version

Personal portfolio: study-abroad information organizer built with Next.js, React, and TypeScript. Includes articles, rule-based `/plan` route planning, local mock `/ai` Q&A MVP, and multilingual `/case-study`. Solo work: UI, business logic, docs, and Vercel deployment. Demonstrates frontend engineering and clear product boundaries—not a commercial service.

### Detailed Version

**Project:** Study Abroad Information Organizer (Personal Portfolio)

**Purpose:** A portfolio project for Japan IT job applications. Organizes study-abroad information into structured pages and demonstrable flows instead of scattered content.

**Stack:** Next.js App Router, React, TypeScript, Tailwind CSS, Git / GitHub, Vercel

**Features:**
- Article system with categories, search, and localized content
- `/plan`: explainable route suggestions from local TypeScript rules
- `/ai`: structured Q&A MVP with mock logic (no real AI API)
- `/case-study`: interview-oriented project walkthrough in three languages

**My role:** End-to-end solo delivery—page structure, components, `lib/plan` and `lib/ai` logic, i18n, documentation, and deployment.

**Scope limits:** No database, login, or payment; no visa or job guarantees. Built for portfolio demonstration and technical interview discussion.
